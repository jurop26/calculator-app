# 📱 iPhone Calculator (React Native + Expo)

Simple iPhone-style calculator built with React Native using Expo.

## ✨ Features

- Basic operations: + − × ÷
- Decimal support (`,` and `.`)
- Toggle sign (+/-)
- Clear (AC) and backspace (⌫)
- Smart formatting (removes trailing zeros)
- Handles negative numbers and parentheses

## 🛠️ Tech Stack

- React Native
- Expo
- TypeScript

## 🚀 Getting Started

```bash
npm install
npx expo start
```
